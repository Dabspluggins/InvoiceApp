# IMPORTANT: Dark mode is required on all new UI
Any new feature pages or components MUST include `dark:` variants for all Tailwind classes. Use `dark:bg-gray-800` for cards, `dark:bg-gray-900` for page backgrounds, `dark:text-white` for headings, `dark:text-gray-300` for body text, `dark:border-gray-700` for borders. See the dark mode class mapping in the estimates components for the full reference table.
